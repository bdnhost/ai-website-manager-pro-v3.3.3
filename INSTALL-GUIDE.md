# 🤖 AI Website Manager Pro v3.3.0 - מדריך התקנה

## 📋 דרישות מערכת

- WordPress 5.0 ומעלה
- PHP 7.4 ומעלה
- הרשאות ניהול (Administrator)

## 🚀 התקנה מהירה

### שיטה 1: דרך לוח הבקרה של WordPress

1. היכנס ללוח הבקרה של WordPress
2. עבור אל **תוספים** → **הוסף חדש**
3. לחץ על **העלה תוסף**
4. בחר את הקובץ `ai-website-manager-pro-v3.3.0.zip`
5. לחץ **התקן עכשיו**
6. לחץ **הפעל תוסף**

### שיטה 2: העלאה ידנית דרך FTP

1. חלץ את הקובץ `ai-website-manager-pro-v3.3.0.zip`
2. העלה את התיקייה `ai-website-manager-pro` לתיקייה `/wp-content/plugins/`
3. היכנס ללוח הבקרה של WordPress
4. עבור אל **תוספים** → **תוספים מותקנים**
5. מצא את "AI Website Manager Pro" ולחץ **הפעל**

## ⚙️ הגדרה ראשונית

### 1. הגדרת מפתחות API

לאחר ההפעלה, עבור אל:
**AI Manager Pro** → **🔑 מפתחות API**

הוסף לפחות מפתח API אחד:

- **OpenAI**: `sk-...` (מומלץ)
- **Anthropic**: `sk-ant-...`
- **OpenRouter**: `sk-or-...` (גישה למודלים מרובים)
- **DeepSeek**: `sk-...` (חדש!)

### 2. יצירת מותג ראשון

עבור אל: **AI Manager Pro** → **🏢 ניהול מותגים**

- הוסף מותג חדש עם פרטי העסק שלך
- או ייבא מותג מקובץ JSON

### 3. התחלת יצירת תוכן

עבור אל: **AI Manager Pro** → **🤖 יצירת תוכן**

- בחר מותג פעיל
- בחר סוג תוכן
- הזן נושא
- לחץ "צור תוכן"
