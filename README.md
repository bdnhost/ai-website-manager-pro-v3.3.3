## 🚀 התקנה מהירה

### שיטה 1: דרך ממשק הניהול של WordPress (מומלץ)

1. **הורד את הפלאגין**
   - הורד את קובץ `ai-website-manager-pro-v3.2.1.zip`
   - שמור את קובץ ה-ZIP ללא חילוץ

2. **העלה דרך ממשק הניהול**
   - התחבר לממשק הניהול של WordPress
   - נווט ל**תוספים → הוסף חדש**
   - לחץ על **העלה תוסף**
   - בחר את קובץ ה-ZIP ולחץ **התקן עכשיו**
   - לחץ **הפעל תוסף** כשההתקנה מסתיימת

3. **הגדרה ראשונית**
   - תראה "AI Manager Pro" בתפריט הניהול
   - לחץ עליו כדי לגשת לממשק ה-SPA המודרני
   - עקב אחר אשף ההגדרה לקביעת ההגדרות

### שיטה 2: העלאה דרך FTP/מנהל קבצים

1. **חלץ קבצים**
   - חלץ את קובץ ה-ZIP למחשב שלך
   - אמור לראות תיקייה בשם `ai-website-manager-pro`

2. **העלה דרך FTP**
   - התחבר לאתר שלך דרך FTP או מנהל קבצים
   - נווט ל`/wp-content/plugins/`
   - העלה את כל תיקיית `ai-website-manager-pro`

3. **הפעל תוסף**
   - עבור לממשק הניהול של WordPress → תוספים
   - מצא "AI Website Manager Pro" ולחץ **הפעל**

## ⚙️ הגדרה ראשונית

### שלב 1: הגדרת מפתחות API

1. **גישה להגדרות**
   - עבור ל**AI Manager Pro** בתפריט הניהול
   - נווט ל**הגדרות → מפתחות API** (או השתמש בממשק ה-SPA)

2. **הוסף מפתחות API**

   ```
   מפתח OpenAI: sk-...your-openai-key...
   מפתח Anthropic: sk-ant-...your-anthropic-key... (אופציונלי)
   מפתח OpenRouter: sk-or-...your-openrouter-key... (אופציונלי)
   ```

3. **בדוק חיבורים**
   - לחץ **בדוק חיבור** לכל מפתח API
   - וודא שכל החיבורים מציגים סטטוס "מחובר"

### שלב 2: הגדרת מותג

1. **צור את המותג הראשון שלך**
   - נווט ל**ניהול מותגים**
   - לחץ **צור מותג חדש** או **בדוק עורך JSON**

2. **הגדר הגדרות מותג**

   ```json
   {
     "name": "שם המותג שלך",
     "description": "תיאור המותג",
     "voice": {
       "tone": "מקצועי",
       "style": "אינפורמטיבי",
       "personality": "ידידותי"
     },
     "guidelines": {
       "content_length": "בינוני",
       "keywords": ["המילים", "המפתח", "שלך"],
       "avoid_words": ["הימנע", "ממילים", "אלה"]
     }
   }
   ```

3. **שמור והפעל**
   - השתמש בעורך ה-JSON להתאמת המותג שלך
   - לחץ **שמור הגדרות**
   - הפעל את המותג ליצירת תוכן

### שלב 3: בדיקת יצירת תוכן

1. **גישה למחולל התוכן**
   - נווט ל**מחולל תוכן** בממשק ה-SPA
   - או השתמש בתפריט המסורתי: **AI Manager Pro → מחולל תוכן**

2. **צור תוכן בדיקה**
   - בחר סוג תוכן (למשל, "פוסט בלוג")
   - בחר את קול המותג שלך
   - הזן נושא בדיקה
   - לחץ **צור תוכן**

3. **אמת תוצאות**
   - התוכן אמור להיווצר תוך 10-30 שניות
   - בדוק שקול המותג מיושם נכון
   - בדוק סוגי תוכן ואורכים שונים

## 🎨 התאמה אישית

### שילוב עם ערכת עיצוב

ממשק ה-SPA משתמש בסגנונות הניהול של WordPress ומתאים לערכת הצבעים של הניהול שלך.

### עיצוב מותאם

הוסף CSS מותאם כדי לעקוף סגנונות ברירת מחדל:

```css
/* סגנונות SPA מותאמים */
.ai-spa-container {
  /* הסגנונות המותאמים שלך */
}

.ai-dashboard-container {
  /* התאמות דשבורד */
}
```

### הרחבות JavaScript

הרחב את פונקציונליות ה-SPA:

```javascript
// גישה לנתב SPA
if (window.aiSpaRouter) {
  // הוסף ניווט מותאם
  aiSpaRouter.navigateTo("custom-route");

  // בדוק נתיב נוכחי
  const currentRoute = aiSpaRouter.getCurrentRoute();
}
```
