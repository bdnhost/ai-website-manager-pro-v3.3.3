# AI Website Manager Pro v3.3.5

## תיאור
מערכת ניהול תוכן מתקדמת עם בינה מלאכותית המאפשרת יצירת תוכן איכותי, ניהול מותגים חכם ואוטומציה מלאה של תהליכי התוכן.

## תכונות עיקריות

### 🎨 דשבורד מודרני משופר (v3.3.4+)
- תצוגה חדשנית עם קלפים אינטראקטיביים
- סטטיסטיקות בזמן אמת
- ניהול מותג חכם משולב
- מעקב אחר ביצועים

### 🏢 ניהול מותגים מתקדם (v3.3.5)
- **תצוגת מותג מלאה** - modal מפורט עם כל פרטי המותג
  - מידע בסיסי (שם, תעשייה, טון דיבור)
  - מילות מפתח וערכים עם תגיות צבעוניות
  - קהל יעד, משימה, חזון ו-USP
  - קישורים ומידע אינטרנטי
- ייבוא/ייצוא מותגים בפורמט JSON
- 10+ דוגמאות מותגים מוכנות
- ממשק עברית מלא

### 🤖 Smart Brand Content Engine
- **Topic Pool** - מאגר נושאים מתוחכם
- **AI Topic Generator** - יצירת נושאים אוטומטית
- **RSS Trending Detector** - זיהוי טרנדים מ-RSS
- **Performance Analytics** - ניתוח ביצועים
- **Content Gap Analyzer** - זיהוי פערי תוכן

### 💡 סנכרון חכם עם יצירת תוכן (v3.3.5 חדש!)
- **טעינה דינמית** של כל המותגים מה-DB
- **סימון אוטומטי** של המותג הפעיל
- **הצעות תוכן חכמות** מבוססות:
  - מילות מפתח של המותג
  - תעשיית המותג
  - קהל היעד
  - הצעת הערך הייחודית (USP)
- **פאנל הצעות אינטראקטיבי** עם:
  - עד 10 הצעות לפוסטים
  - לחיצה על הצעה ממלאת אוטומטית את הטופס
  - אנימציות ו-hover effects
  - תצוגת מידע מותג עם מילות מפתח

### ✍️ יצירת תוכן עם AI
- תמיכה במודלים מובילים:
  - OpenAI (GPT-3.5, GPT-4)
  - Anthropic (Claude)
  - **DeepSeek LLM** (תמיכה מלאה)
  - Google (Gemini)
  - Meta (Llama)
- תבניות SEO אוטומטיות
- 25+ פרומפטים מתקדמים
- ספריית פרומפטים

### 🔄 אוטומציה מתקדמת
- תזמון משימות אוטומטי
- יצירת תוכן מתוזמן
- פרסום אוטומטי
- דוחות וניתוחים

### 📊 ניתוח וביצועים
- מעקב אחר ביצועי תוכן
- ניתוח SEO אוטומטי
- המלצות לשיפור
- דוחות מפורטים

## דרישות מערכת
- WordPress 5.0 ומעלה
- PHP 7.4 ומעלה
- MySQL 5.6 ומעלה

## התקנה
1. העלה את התיקייה לספריית `/wp-content/plugins/`
2. הפעל את התוסף דרך תפריט 'Plugins' בוורדפרס
3. הגדר את מפתחות ה-API בהגדרות

## שינויים בגרסה 3.3.5

### ✨ חדש
- **תצוגת מותג מלאה** - כפתור "View" חדש בדף המותגים
- **סנכרון מותג-תוכן** - אינטגרציה מלאה בין מותגים ליצירת תוכן
- **הצעות תוכן אוטומטיות** - מערכת הצעות חכמה מבוססת פרופיל המותג
- **עיצוב מודרני** - UI/UX משופר עם אנימציות חלקות

### 🔧 שיפורים
- ממשק משתמש אינטואיטיבי יותר
- ביצועים מהירים יותר
- תמיכה מלאה ב-RTL (עברית)
- הודעות שגיאה ברורות יותר

### 🐛 תיקוני באגים
- תיקון שגיאת `saveAutomationTask is not defined`
- שיפור טיפול בשגיאות AJAX
- תיקון בעיות תצוגה במסכים קטנים
