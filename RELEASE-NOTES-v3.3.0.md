# AI Website Manager Pro v3.3.0 - Professional Release

## 🚀 סקירת גרסה

גרסה 3.3.0 מביאה שיפורים מהותיים שהופכים את הפלאגין למערכת ניהול תוכן מקצועית ומלאה עם בינה מלאכותית.

## ✨ תכונות חדשות מרכזיות

### 1. 🎯 דשבורד מקצועי חדש לגמרי

- **ממשק אינטואיטיבי** עם סטטיסטיקות בזמן אמת
- **מדריך התחלה מהירה** המנחה משתמשים חדשים
- **מצב מערכת חכם** הבודק תקינות ספקי AI ומותגים
- **פעולות מהירות** לגישה מיידית לכל הפונקציות
- **עיצוב רספונסיבי** מותאם לכל המכשירים

### 2. 🏢 ניהול מותגים מתקדם עם JSON

- **ייבוא מותגים מ-JSON** - העלאת מותגים מקצועיים
- **ייצוא מותגים ל-JSON** - גיבוי ושיתוף מותגים
- **תבנית JSON מקצועית** עם כל השדות הנדרשים:
  - מידע מותג מפורט
  - קול וטון תקשורת
  - קהל יעד ופרסונות
  - הנחיות תוכן מתקדמות
  - מילות מפתח ומסרים
  - זהות ויזואלית
  - דרישות רגולטוריות
  - לוח תוכן ומדדי ביצוע

### 3. 📚 מערך פרומפטים מקצועי מורחב (25+ פרומפטים)

#### קטגוריות קיימות משופרות:

- **שיווק ופרסום** (3 פרומפטים)
- **יצירת תוכן** (3 פרומפטים)
- **עסקים ומקצועי** (3 פרומפטים)
- **יצירתי ומעורר השראה** (3 פרומפטים)
- **טכני ומקצועי** (4 פרומפטים)

#### קטגוריות חדשות:

- **🤖 מתמחה בבינה מלאכותית** (3 פרומפטים)
  - פרומפט אופטימלי ל-DeepSeek
  - הנדסת פרומפטים מתקדמת
  - אנליזה והשוואת מודלי AI

- **🔍 SEO ואופטימיזציה מתקדמת** (3 פרומפטים)
  - תוכן SEO מותאם לכוונת חיפוש
  - אסטרטגיית תוכן לקלסטר נושאי
  - אופטימיזציה לחיפוש קולי

- **📈 אופטימיזציית המרות** (3 פרומפטים)
  - דף נחיתה בעל המרה גבוהה
  - אימייל נטישת עגלה
  - טקסט לכפתור קריאה לפעולה

### 4. 🧠 תמיכה מלאה ב-DeepSeek LLM

- **2 מודלים זמינים**: DeepSeek Chat ו-DeepSeek Coder
- **בחירה אוטומטית** של מודל לפי סוג התוכן
- **מעקב עלויות** וחישוב משוער
- **סטטיסטיקות שימוש** מפורטות

## 🚀 התקנה ושדרוג

### דרישות מערכת

- WordPress 5.0 ומעלה
- PHP 7.4 ומעלה
- זיכרון: 256MB מומלץ
- מפתח API לפחות מספק AI אחד

### הוראות התקנה

1. הורד את `ai-website-manager-pro-v3.3.0-professional.zip`
2. התקן דרך WordPress Admin → תוספים → הוסף חדש
3. הפעל את הפלאגין
4. עקוב אחר מדריך ההתחלה המהירה בדשבורד
