# שילוב DeepSeek LLM ב-AI Website Manager Pro

## סקירה כללית

הוספתי תמיכה מלאה ב-DeepSeek LLM למערכת AI Website Manager Pro. השילוב כולל:

### תכונות חדשות:

- **שירות DeepSeek מלא** - מחלקה ייעודית לטיפול ב-API של DeepSeek
- **2 מודלים זמינים**:
  - `deepseek-chat` - מודל שיחה מתקדם לתוכן כללי
  - `deepseek-coder` - מודל מיוחד לכתיבת קוד ותיעוד טכני
- **בחירה חכמה של מודל** - המערכת בוחרת אוטומטית את המודל המתאים לסוג התוכן
- **מעקב עלויות** - חישוב עלויות משוער לפני יצירת תוכן
- **סטטיסטיקות שימוש** - מעקב מפורט אחר השימוש במודלי DeepSeek

### קבצים שנוספו:

1. `includes/ai/class-deepseek-service.php` - שירות DeepSeek הראשי
2. `includes/ajax/class-deepseek-ajax.php` - טיפול בבקשות AJAX

### קבצים שעודכנו:

1. `ai-website-manager-pro.php` - הוספת תמיכה ב-DeepSeek בקובץ הראשי
2. `includes/ai/class-ai-service.php` - הוספת DeepSeek לרשימת הספקים
3. `includes/ai/class-real-ai-service.php` - תמיכה ב-DeepSeek בשירות האמיתי

## הוראות שימוש:

### 1. הגדרת מפתח API:

- עבור ל**AI Manager Pro → מפתחות API**
- הזן את מפתח ה-API של DeepSeek בשדה המתאים
- לחץ "בדוק חיבור" לוודא שהמפתח תקין

### 2. בחירת DeepSeek כספק ברירת מחדל:

- עבור ל**AI Manager Pro → הגדרות כלליות**
- בחר "DeepSeek" בתפריט "ספק AI ברירת מחדל"
- בחר את המודל המועדף (Chat או Coder)

### 3. יצירת תוכן:

- המערכת תבחר אוטומטית את המודל המתאים
- DeepSeek Coder ישמש לתיעוד טכני וקוד
- DeepSeek Chat ישמש לתוכן כללי

## יתרונות DeepSeek:

- **מחירים תחרותיים** - $0.14 לכל מיליון טוקנים קלט
- **ביצועים מעולים** - איכות תוכן גבוהה
- **התמחות בקוד** - מודל Coder מיוחד לפיתוח
- **תמיכה בעברית** - יצירת תוכן איכותי בעברית

השילוב מוכן לשימוש מיידי!
