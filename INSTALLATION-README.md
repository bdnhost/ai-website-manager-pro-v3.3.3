# AI Website Manager Pro v3.2.2 - עם תמיכה ב-DeepSeek LLM

## 🚀 התקנה מהירה

### דרישות מערכת

- WordPress 5.0+
- PHP 7.4+
- זיכרון: 128MB מינימום

### שלבי התקנה

1. **הורד והתקן**
   - הורד את `ai-website-manager-pro-v3.2.2-with-deepseek.zip`
   - עלה דרך WordPress Admin → תוספים → הוסף חדש → העלה תוסף
   - הפעל את הפלאגין

2. **הגדרת מפתחות API**
   - עבור ל-AI Manager Pro → מפתחות API
   - הזן מפתחות עבור הספקים הרצויים:
     - OpenAI: `sk-...`
     - Anthropic: `sk-ant-...`
     - OpenRouter: `sk-or-...`
     - **DeepSeek: `sk-...`** (חדש!)

3. **בחירת ספק ברירת מחדל**
   - עבור להגדרות כלליות
   - בחר את הספק המועדף (כולל DeepSeek החדש)
