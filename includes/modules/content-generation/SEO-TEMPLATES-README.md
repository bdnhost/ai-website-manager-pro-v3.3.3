# מערכת תבניות SEO - מדריך שימוש

## סקירה כללית

מערכת תבניות SEO מספקת מבנה אוטומטי ואופטימלי לכל תוכן שנוצר במערכת. כל פוסט שנוצר יקבל אוטומטית מבנה מושלם של כותרות, טבלאות, רשימות, ואלמנטים נוספים הנדרשים ל-SEO מתקדם.

## תכונות עיקריות

### ✅ מבנה כותרות היררכי
- כותרת H1 אחת בלבד לכל תוכן
- כותרות H2 למדורים ראשיים (3-5 כותרות)
- כותרות H3 לתת-מדורים (6-15 כותרות)
- כותרות H4 לתת-תת-מדורים (לפי הצורך)

### ✅ אלמנטי תוכן עשירים
- **טבלאות HTML** - לוחות השוואה, מפרטים טכניים
- **רשימות מסודרות** - הוראות צעד אחר צעד
- **רשימות לא מסודרות** - תכונות, יתרונות, חסרונות
- **תוכן עניינים** - קישורי עוגן למדורים עיקריים
- **סעיף FAQ** - שאלות ותשובות נפוצות

### ✅ תמיכה ב-4 סוגי תוכן שונים

#### 1. Article (מאמר)
מאמר מקיף ומעמיק עם:
- הקדמה מושכת
- תוכן עניינים אינטראקטיבי
- 3-5 מדורים ראשיים
- טבלאות השוואה
- סעיף FAQ
- סיכום ומסקנות

**דוגמת מבנה:**
```
H1: כותרת המאמר
  → הקדמה (2-3 משפטים)
  → תוכן עניינים

H2: מדור ראשי 1
  H3: תת-מדור 1.1
  H3: תת-מדור 1.2

H2: מדור ראשי 2
  H3: תת-מדור 2.1
  H3: תת-מדור 2.2
  → טבלה להשוואה

H2: שאלות נפוצות (FAQ)
  H3: שאלה 1?
  H3: שאלה 2?

H2: סיכום
```

#### 2. Guide (מדריך הדרכה)
מדריך צעד אחר צעד עם:
- הקדמה - מה תלמדו
- "מה תצטרכו" - דרישות מקדימות
- צעדים ממוספרים (H3 לכל צעד)
- הערכות זמן לכל שלב
- טעויות נפוצות שיש להימנע מהן
- טיפים מקצועיים
- FAQ

**דוגמת מבנה:**
```
H1: איך לבצע [משימה]: מדריך מלא
  → הקדמה
  → תוכן עניינים

H2: מה תצטרכו
  → רשימת כלים וחומרים

H2: מדריך צעד אחר צעד
  H3: צעד 1: [פעולה ראשונה]
    → הוראות מפורטות
    → תת-צעדים (OL)
    → ⏱️ זמן משוער: 10 דקות

  H3: צעד 2: [פעולה שנייה]
    → הוראות מפורטות

H2: טעויות נפוצות שכדאי להימנע מהן
  → רשימת טעויות (UL)

H2: טיפים מקצועיים
  → טבלה עם טיפים

H2: שאלות נפוצות
H2: סיכום
```

#### 3. Review (ביקורת מוצר/שירות)
ביקורת מקיפה עם:
- סקירה מהירה עם טבלת מפרטים
- יתרונות וחסרונות (טבלה דו-עמודתית)
- ניתוח מפורט של תכונות
- השוואה עם אלטרנטיבות
- דירוגים וציונים
- המלצה סופית

**דוגמת מבנה:**
```
H1: ביקורת [מוצר] 2025: האם זה שווה את ההשקעה?
  → הקדמה עם פסק דין מהיר

H2: סקירה מהירה
  → טבלת מפרטים
  ├─ שם המוצר
  ├─ קטגוריה
  ├─ מחיר
  ├─ דירוג: ★★★★☆ (4/5)
  └─ מתאים ל...

H2: יתרונות וחסרונות
  → טבלה עם 2 עמודות:
  ├─ יתרונות ✅
  └─ חסרונות ❌

H2: ניתוח מפורט
  H3: תכונה 1
    → ניתוח + דירוג (8/10)
  H3: תכונה 2
    → ניתוח + דירוג (7/10)

H2: השוואה עם מתחרים
  → טבלת השוואה מקיפה

H2: למי זה מתאים?
  → מתאים במיוחד ל...
  → לא מומלץ ל...

H2: מחירים ותוכניות
  → טבלת תוכניות ומחירים

H2: פסק הדין שלנו
  → דירוג כולל: ★★★★☆ (4.2/5)
  ├─ תכונות: 8.5/10
  ├─ קלות שימוש: 9/10
  ├─ תמורה למחיר: 7.5/10
  └─ תמיכה: 8/10

H2: שאלות נפוצות
H2: מחשבות אחרונות
```

#### 4. Product (תיאור מוצר)
תיאור מוצר למכירה עם:
- כותרת משכנעת עם USP
- תכונות עיקריות
- מה כלול באריזה
- תכונות מפורטות
- מפרטים טכניים
- תרחישי שימוש
- השוואה עם מתחרים
- אלמנטי אמון (אחריות, גרנטי)

**דוגמת מבנה:**
```
H1: [שם מוצר] - [הבטחה עיקרית]
  → תיאור מושך (2-3 משפטים)

H2: תכונות עיקריות במבט חטוף
  ✅ תכונה 1: יתרון
  ✅ תכונה 2: יתרון
  ✅ תכונה 3: יתרון

H2: מה כלול באריזה
  → רשימת פריטים

H2: תכונות מפורטות
  H3: תכונה 1
    → מה זה עושה
    → למה זה חשוב
  H3: תכונה 2
    → מה זה עושה
    → למה זה חשוב

H2: מפרטים טכניים
  → טבלה עם מפרטים:
  ├─ מידות
  ├─ משקל
  ├─ חומרים
  ├─ אפשרויות צבע
  └─ אחריות

H2: תרחישי שימוש בחיים האמיתיים
  H3: שימוש 1
  H3: שימוש 2

H2: מה מייחד אותו
  → טבלת השוואה עם מתחרים

H2: למי כדאי לקנות
  → מתאים במיוחד ל...
  → לא מומלץ ל...

H2: ערבות לאיכות
  ✅ אחריות X שנים
  ✅ החזר כספי תוך 30 יום
  ✅ אישורי איכות
  ✅ תמיכה 24/7

H2: שאלות נפוצות
H2: מוכנים להתחיל?
```

## איך להשתמש

### שימוש בסיסי

כאשר אתם יוצרים תוכן חדש, פשוט בחרו את סוג התוכן:

```php
$content_generator->generate_content([
    'topic' => 'הדרכה מלאה לשיווק דיגיטלי',
    'content_type' => 'guide',  // article, guide, review, product
    'brand_id' => 1,
    'auto_publish' => true,
    'post_status' => 'draft'
]);
```

המערכת **אוטומטית**:
1. ✅ תוסיף הוראות SEO מפורטות לפרומפט
2. ✅ תיצור תוכן עם המבנה המתאים
3. ✅ תוודא שהתוכן עומד בתקני SEO
4. ✅ תדווח על ציון SEO ובעיות אפשריות

### ולידציה אוטומטית

לאחר יצירת התוכן, המערכת בודקת:
- מספר כותרות H1 (חייבת להיות 1 בדיוק)
- מספר כותרות H2, H3, H4
- קיום טבלאות (אם נדרש)
- קיום רשימות
- קיום FAQ (אם נדרש)
- קיום תוכן עניינים (אם נדרש)

**תוצאת הולידציה:**
```php
[
    'score' => 95,  // ציון SEO (0-100)
    'passed' => true,  // האם עבר את הדרישות המינימליות
    'issues' => [],  // רשימת בעיות שנמצאו
    'template_used' => 'Article'  // התבנית שהשתמשו בה
]
```

### הוספת תבנית מותאמת אישית

ניתן להוסיף תבניות נוספות:

```php
$seo_template_manager->add_custom_template('my_template', [
    'name' => 'התבנית שלי',
    'description' => 'תיאור התבנית',
    'structure' => [
        'introduction' => true,
        'main_sections' => 3,
        // ...
    ],
    'seo_prompt_additions' => '
    הוראות SEO מפורטות כאן...
    ',
    'heading_structure' => [
        'h1' => 1,
        'h2' => '3-5',
        'h3' => '6-12',
    ],
    'content_elements' => [
        'tables' => true,
        'lists' => true,
        // ...
    ]
]);
```

- תוכן אופטימלי ל-SEO ללא מאמץ
- מבנה מקצועי לכל פוסט
- שיפור דירוג במנועי חיפוש
- חיסכון בזמן עריכה

### 🤖 ל-AI
- הוראות ברורות ומפורטות
- פורמט אחיד לכל התוכן
- פחות שגיאות במבנה
- תוצאות עקביות

### 📊 ל-SEO
- כותרות היררכיות תקינות
- טבלאות עשירות לתוכן
- FAQ מובנה
- תוכן עניינים לניווט
- מבנה ידידותי למנועי חיפוש

## API למפתחים

### קבלת תבנית
```php
$template_manager = new SEO_Template_Manager();
$template = $template_manager->get_template('article');
```

### שיפור פרומפט
```php
$enhanced_prompt = $template_manager->enhance_prompt_with_seo(
    $original_prompt,
    'guide'
);
```

### ולידציה של תוכן
```php
$validation = $template_manager->validate_seo_structure(
    $content,
    'review'
);

if (!$validation['passed']) {
    echo "ציון SEO: " . $validation['score'];
    print_r($validation['issues']);
}
```

### קבלת כל התבניות
```php
$all_templates = $template_manager->get_all_templates();
foreach ($all_templates as $name => $template) {
    echo $template['name'] . ": " . $template['description'];
}
```
